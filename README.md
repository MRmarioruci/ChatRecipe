# ChatRecipe
A fun project in which users can generate recipes based on what they have on their inventory. Let AI do the work, sit back and enjoy different recipe suggestions.

<b>Technologies</b>
<ul>
  <li>Python</li>
  <li>Flask</li>
  <li>Mysql</li>
  <li>React.js</li>
  <li>Typescript</li>
  <li>Scss</li>
  <li>Rest APIs</li>
  <li>ChatGPT</li>
</ul>
